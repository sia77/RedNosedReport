# 🎄 Red-Nosed Reports Analyzer

A React + TypeScript tool to solve the Advent of Code Day 2 challenge. 

## 🚀 Features
- **Data Analysis:** Calculates if reports are safe based on increasing/decreasing levels.
- **Flexbox UI:** Responsive layout with sticky bottom upload controls.
- **Unique Tracking:** Uses `uuid` for stable React reconciliation.

## 🛠️ Installation
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
